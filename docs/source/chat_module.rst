Chat Module
===========

.. automodule:: routers.chat
  :members:
  :undoc-members:
  :show-inheritance:

This is the **core AI engine** for handling user queries.

Key Features
------------

- Accepts user messages
- Enqueues requests for AI processing
- Supports follow-up conversations
- Displays AI response after queue completion

Integrated with:
- `queue_module`
- `notifications_module`
